- 🎯 **背景**: 日本のアニメ会社の収益構造は通常「制作委員会方式」で、ヒットが出てもリスクと共に利益が出資比率で分散される。東映は主要IPの版権を自社で独占的に保有・コントロールしている点で他社と決定的に異なり、海外でゲーム化やライセンスアウトが進めば、その粗利がほぼまるごと自社に落ちる。つまり「制作スタジオ」ではなく「IPホルダー」として読むべき会社。

## 2. ジーダット (3841) — 「AI関連デバイスは好調、アナログ・パワーは横ばい」二極化の歪みに食い込むEDAツール株

- 🎯 **背景**: 半導体市場の中でも「AI関連デバイス（HBM、GPU、AIアクセラレータ）は爆発的に伸びる一方、アナログ・パワーデバイスは安定だが大きな伸びは期待しにくい」という二極化が進んでいる。普通に考えるとアナログ・パワー寄りのEDAは退屈に見えるが、伸びにくい分野だからこそ各社は設計工数・人件費を削るためにAI支援の設計自動化ツールへの依存度を高めざるを得ない。ジーダットはちょうどそこに刺さるニッチプレイヤー。
- 📌 **根拠**:

  - 自社で語る市場観として「AIに関連するデバイスは非常に好調、アナログデバイス・パワーデバイスはステーブルで厳しい状況」と二極化を明示
- 💡 **着眼点**: 大型AI関連株は既に高PERで買われているが、「AI設計を支える側」のEDAは表舞台にあまり出てこない。しかも市場全体が二極化している局面こそ、伸びにくい分野（アナログ・パワー）のメーカーが自動化ツールに依存を強めるという、典型的な「ピックアンドシャベル」構造が成立する。中国市場の慎重姿勢が長引いている点はリスクだが、国産EDAは安全保障文脈でも意識される可能性がある。

## 3. ウォルマート (WMT) 売り — 燃料高で低所得層が「シリアルしか買えない」状態に追い詰められると粗利率が崩れる
- 📺 **動画**: [OpenAI上場＆アーム株価急騰→ソフトバンク！サンディスクはバブル継続. エヌビディア好決算でもなぜ売るのか？ ダウ最高値更新はやはりAI株](https://www.youtube.com/watch?v=Dol4fvmKHmA) / 米国株投資マット / 2026-05-22 07:14 / 再生498回
- 💰 **アイディア**: 米国生活必需品セクターの代表格ウォルマート(WMT)に対し、短期（〜3ヶ月）の空売りまたはアンダーウェイト方針を取る。来店客数や売上高そのものではなく、四半期ごとの「ミックスシフト（粗利率の悪い低価格食料品の構成比上昇）」と「同店粗利率」の悪化をトリガーとして監視し、それが確認されたら売り増しする。
- 🎯 **背景**: 一般には「燃料高・物価高ではディスカウンターのウォルマートに客が流れてプラス」とポジティブに語られがちだが、実態は逆方向に進みつつある。来店客は確かに増えているが、その中身は「コストパフォーマンスを重視する低所得層が、利益率の悪い低価格な食料品（シリアル等の必需品）しか買ってくれない」という構成。トップライン上は売上が立っているように見えても、粗利率の低い商品の構成比が上がるほど営業利益はじわじわと削られる。
- 📌 **根拠**:
  - 動画内で「燃料費の高騰により、コストパフォーマンスを重視する消費者が同社の低価格な食料品や生活必需品に殺到」「いわゆるアラリ率が悪い低価格の食料品、シリアルとかそういったものしか買ってくれない」「ここから低所得者にまず影響が及んでいます」と具体的な購買行動の変質を指摘
  - 過去のインフレ局面で同じ「客は増えるがバスケットが薄くなる」現象が利益率の圧迫として顕在化した事例の延長線
- 💡 **着眼点**: 「インフレ・燃料高はディスカウンター強気」という条件反射の裏側で、客層の質と購買バスケットの中身が静かに劣化している。決算ヘッドラインの売上・客数だけ見ると好印象でも、ミックスシフトを丁寧に読むと利益率が削られている、というギャップが取れる局面。同様の構造は他のディスカウントストアにも応用できるが、最も売上規模が大きく市場の「逃避先」と見なされているウォルマートで歪みが先に出る可能性が高い。

## 4. 国策小型株は「主役の防衛・宇宙大手」より「その下のセンサー・電源・素材サプライヤー」を狙う
- 📺 **動画**: [第二の三菱重工はどこか？国策小型株で化ける銘柄と危ない銘柄の決定的な差](https://www.youtube.com/watch?v=sBmLfmDdjJo) / 株の裏側 / 2026-05-22 14:53 / 再生68回
- 💰 **アイディア**: 防衛・宇宙・エネルギー・半導体・電力インフラ・サイバーセキュリティといった国策分野について、装備や完成品を作る大手ではなく、その下のサプライチェーン（特殊部品・センサー・素材・精密加工・通信機器・電源装置・組込ソフトウェア）を担う日本の小型銘柄を狙う。決算で「受注残の積み上がり」と「セグメント利益率の改善」という地味な数字が出始めた銘柄に、複数回に分けてエントリーする。
- 🎯 **背景**: 国策テーマで主役級の大型株（三菱重工、IHIなど）は既に大きく買われている。一方、これらが国から受注した予算は、最終的にサプライチェーンの末端にある小型部品メーカーに流れていく。市場は完成品メーカーには素早く反応するが、その下流の小型サプライヤーが受注残・利益率という形で実態を見せ始めるまでにタイムラグがある。ここを取りに行く。
- 📌 **根拠**:
  - 「大企業の周辺で、さらに欠かせない技術や部品を持つ企業こそ、市場に気づかれた瞬間に評価が大きく変わる」
  - 失敗パターンとして「期待が先行して、まだ利益が大きく出ていない段階で将来の夢だけで買われる→SNSで話題→出来高急増→少しでも材料が出なくなると一気に売られる」を明示
  - 成功条件は「決算資料で地味な変化（受注増、利益率改善）を確認し、実態が伴っているかを見極める」「業界や受注の確認をしながら、何回かに分けて資金を投入する」
- 💡 **着眼点**: 「国策テーマ＝買い」という単純な短絡から一歩踏み込み、予算が最終的にどの部品・素材・システムに流れていくかを地図化する発想。「準備＞勇気」「夢で買わず受注残で買う」という運用ルールがセットになっているのが特徴で、テーマ株の急騰急落で資金を溶かしがちな個人投資家にとっての具体的な歯止めにもなる。完成品メーカーと末端サプライヤーで時間軸を分ける、いわばリレー投資の発想。
