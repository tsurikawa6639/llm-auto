# Palladyne AIの「AIソフトウェア転換」成功を前提とした中長期投資

本銘柄は現在、ハードウェア事業から高利益率のAIソフトウェア事業への転換期にあり、財務的には「種まき」の段階です。投資判断としては、表面上の黒字に惑わされず、以下の指標をモニタリングしつつ、株価が調整したタイミングでのエントリーを検討します。
1. **受注残高（Backlog）の推移**: 特に資金手当済み（Funded）の案件が順調に積み上がっているかを確認する。
2. **AIソフトウェアのライセンス収入**: 2026年以降、売上高に占めるソフトウェア収入の割合が拡大し、営業損失が縮小に向かうかを注視する。

## 因果関係
1. ハードウェア事業の縮小とAIソフトウェアへの投資強化により、短期的には営業損失が拡大する。
2. 一方で、自社株価の低迷によりワラント負債の評価益が発生し、会計上は純利益が黒字化する。
3. 投資家がこの「帳簿上の黒字」と「本業の赤字」を正しく区別することで、株価にボラティリティが生じる。
4. 今後、受注残高の増加とライセンス収入の本格化が確認されれば、本業の収益性が改善し、株価の再評価（リレーティング）が期待できる。
